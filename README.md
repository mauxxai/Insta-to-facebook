# 🎭 InstaBook — The Masterpiece Redesign

**InstaBook** transforms the modern, algorithmic chaos of Instagram into the cozy, chronological, and "boxy" masterpiece that was Facebook circa 2010.

Built with pixel-perfect precision by **mauxx AI**, this extension isn't just a theme—it's a full structural overhaul of the Instagram experience.

---

## 💎 Masterpiece Features

- **The 3-Column Era**: A faithful recreation of the classic 3-column desktop layout.
  - **Left**: Permanent News Feed, Messages, and Videos navigation.
  - **Center**: The "Golden Ratio" 520px-constrained News Feed.
  - **Right**: Functional Live Ticker, Birthdays, and Chat widgets.
- **Visual Authenticity**: 
  - **Color Palette**: Exact Hex matches (#3B5998, #E9EBEE, #f7f8fc).
  - **Typography**: Forced Lucida Grande / Segoe UI hierarchy.
  - **Physical UI**: Beveled headers, inset transitions, and Web 2.0 gradients.
- **Interaction Overhaul**:
  - **Text Links**: Instagram's icons are gone. Posts now feature "Like · Comment · Share" blue text links.
  - **The Poke**: A fully functional-feeling "Poke" button for that 2011 nostalgia.
- **Live Ticker**: A real-time activity stream on your home feed showing (mocked) activity to keep your dashboard feeling alive.

---

## 🛠 Installation

1. Clone or download this repository.
2. Open Chrome/Edge and navigate to `chrome://extensions/`.
3. Enable **Developer Mode** (top right).
4. Click **Load Unpacked** and select this folder.
5. Visit [instagram.com](https://instagram.com) and witness the transformation.
